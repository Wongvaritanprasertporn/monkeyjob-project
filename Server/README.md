Version - v16.14.2 (Latest version recommended)

## Project setup
`````
npm install
`````

## Set your sensitive information in your .env file (use the .env.example as an example);
`````
PORT=
APP_TITLE=
DB_URL=
ACCESS_TOKEN_SECRET=
REFRESH_TOKEN_SECRET=
SENDGRID_API_KEY=
SENDGRID_SENDER=
`````

## Run
`````
npm start
`````

## Usage
`````
Open http://localhost:PORT/
`````
